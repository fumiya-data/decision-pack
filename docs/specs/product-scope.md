# プロダクト範囲

## 1. 対象

- 個人開発で継続管理できる業務支援ツール
- 顧客、購入履歴、在庫、需要予測、意思決定支援を一体で扱いたいユースケース

## 2. 提供価値

- 顧客データと購入履歴を整理して、使える業務データへ変換する
- 顧客ごとの次回購入候補を把握できる
- 需要予測をもとに、品目ごとの在庫リスクを把握できる
- 将来需要と資金制約を踏まえた意思決定を支援できる

## 3. 今回の対象範囲

- 顧客CSVの整形と永続化
- 商品品目リストの管理
- 購入履歴の正規化
- 顧客ごとの購入傾向推定
- 品目別在庫一覧と検索
- 需要予測を使った在庫リスクの評価
- GUI と API の再構成
- AWS 前提の実行構造の設計

## 4. 今回の非対象

- POS や EC サービスとのリアルタイム連携
- 自動発注の完全自律化
- 高度な機械学習基盤の先行導入
- 複数テナント対応
- 厳密な権限管理の実装詳細

## 5. 前提

- 既存モジュールは全面的に再編されうる
- 文書が正本であり、コードはあとから追従する
- ローカルで成立させた後に AWS へ移す
